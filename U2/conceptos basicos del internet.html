¿Que es internet?

Red de computadoras interconectadas entre sí a nivel mundial con el objetivo de hacer común información de acceso público

Tipos de proveedore de internet y nombres

Para un servicio de diseño web completo, necesitas una empresa de diseño web que ofrezca servicios de desarrollo, hoting y registro de dominios, con opciones populares como Wix,Hostinger,Godaddy y SiteGround.

¿Que es www?

WWW significa World Wide Web (Red Mundial de Informacón) y es un sistema de información global que permite acceder a páginas web y otros recursos a través de internet mediante navegadores web.

¿Que es carpeta local y carpeta remota?

Una carpeta local es un directorio almacenado directamente en tu propio dispositivo (como tu PC o teléfono), mientras que una carpeta remota es un directorio que se encuentra en otro dispositivo o servidor al que te conectas a través de una red o Internet.

¿Que es FTP?

FTP, o Protocolo de Transferencia de Archivos (File Transfer Protocol), es un conjunto de reglas que permite transferir archivos entre computadoras a través de una red TCP/IP, como Internet.

¿Que es experiencia de usuario?

La Experiencia de Usuario (UX, por sus siglas en inglés) es la percepción integral de una persona al interactuar con un producto, servicio o sistema

¿Que es interfaze de usuario?

Una interfaz de usuario (IU), o User Interface (UI), es el medio o punto de contacto entre una persona y un dispositivo o sistema digital.

¿Que es un DNS?

DNS (Sistema de Nombres de Dominio) es el sistema que traduce los nombres de dominio (como www.google.com) a direcciones IP numéricas (como 74.125.19.147), que las computadoras usan para identificarse y comunicarse en Internet.

¿Que es Hosting?

El hosting, o alojamiento web, es un servicio que alquila un espacio en un servidor para almacenar todos los archivos, datos y contenidos de una página web, permitiendo que esta sea accesible en Internet a cualquier hora y cualquier día.

¿Que es un full stack desinger?

Un full stack designer es un profesional creativo y técnico que abarca todo el proceso de diseño y desarrollo de un producto digital, desde la concepción inicial de la idea hasta la implementación y el lanzamiento
